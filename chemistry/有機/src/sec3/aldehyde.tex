\subsection{アルデヒド基}

\subsubsection*{フェーリング液の還元(フェーリング反応)}
フェーリング液(\ce{CuSO4}，\ce{NaOH}，酒石酸塩の混合物)を還元して酸化銅(I)の赤色沈殿を生じる。
\eeg{
  \ce{2Cu^2+ + 5OH- + RCHO -> Cu2O v + 3H2O + RCOO-}
}
\subsubsection*{銀鏡反応}
アンモニア性硝酸銀(I)溶液(\ce{AgNO3},\ce{NH3}aq)とアルデヒドが反応して銀を生じる。
\eeg{
  \ce{2[Ag(NH3)2]+ + 3OH- + RCHO -> 2Ag + 4NH3 + 2H2O + RCOO-}
}
ギ酸やベンズアルデヒドは銀鏡反応を示すがフェーリング反応は示さない。
